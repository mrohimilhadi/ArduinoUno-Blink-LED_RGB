# ArduinoUno-Blink-LED_RGB
ArduinoUno-Blink-LED_RGB

This project is a basic implementation of RGB LED control using an Arduino Uno-based microcontroller. The system is designed to generate a controlled blinking effect on each color channel (Red, Green, Blue) through digital output pins.

The RGB LED is configured with the following pin mapping:

Red → Pin 13
Green → Pin 12
Blue → Pin 11

Each color channel is activated and deactivated periodically to produce a simple visual effect. This project serves as a foundational exercise for understanding:

GPIO (General Purpose Input/Output) usage
Digital output control
Timing using delay functions
Basic electronic component interfacing
Key Features
Independent control of each color channel (R, G, B)
Simple and easy-to-understand implementation
Suitable for beginners in embedded systems and electronics
Project Objectives

This project aims to provide a fundamental understanding of how a microcontroller interacts with external devices, particularly an RGB LED. It also serves as a starting point for developing more advanced systems such as:

PWM (Pulse Width Modulation) for color variation
Dynamic lighting effects
Integration with sensors or external inputs
Hardware Requirements
Arduino Uno
RGB LED (Common Cathode or Common Anode, depending on the circuit configuration)
Current-limiting resistors (approximately 220Ω – 330Ω)
Jumper wires
